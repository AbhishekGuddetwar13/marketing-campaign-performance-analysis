# Digital Marketing Campaign Performance Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze digital marketing campaign performance. The dashboard provides insights into campaign budgets, conversions, ROI, audience engagement, and platform performance.

The objective of this project is to transform marketing data into actionable business insights that help optimize campaign effectiveness and improve return on investment.

## Business Insights

### Best Performing Platform
Twitter and Facebook generated the highest conversion volumes among all platforms.

### Audience Performance
Young Adults and Teenagers represented the largest share of total conversions.

### Conversion Trend
Campaign conversions showed a decline from January to March, indicating opportunities for campaign optimization.

### Marketing ROI
The dashboard helps identify high-performing campaigns and channels delivering the best return on investment.
